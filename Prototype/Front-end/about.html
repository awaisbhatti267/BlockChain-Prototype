About DoubleForge

DoubleForge is a blockchain simulation tool designed to test
and visualize double-spending attacks on a simulated blockchain
network. It helps learners and researchers understand
vulnerabilities in blockchain consensus mechanisms and explore
possible countermeasures.

This tool is developed using Python, HTML, CSS, and JavaScript.
It includes features like transaction creation, network
simulation, attack testing, and log visualization.

The goal of DoubleForge is to educate users about blockchain
attacks in a visual and interactive way.
